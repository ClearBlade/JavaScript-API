REST API URL Documentation

Fetch Collection
	URL
		https://platform.clearblade.com/apidev/<collectionID>
	Type
		GET
	Headers
		CLEARBLADE-APPKEY:<key>
		CLEARBLADE-APPSECRET:<secret>

Create Item in Collection
	URL
		https://platform.clearblade.com/apidev/<collectionID>
	TYPE
		POST
	Headers
		CLEARBLADE-APPKEY:<key>
		CLEARBLADE-APPSECRET:<secret>
	Data (all columns must exist, not required to specify all columns)
		JSON OBJECT ex. {colName1:value,colName2:value}

Update
	URL
		https://platform.clearblade.com/apidev/<collectionID>
	TYPE
		POST
	Headers
		CLEARBLADE-APPKEY:<key>
		CLEARBLADE-APPSECRET:<secret>
	Data (all columns must exist, not required to specify all columns)
		JSON OBJECT ex. {query:<queryJSONObj>,$set:<changeJSONObj>}

Delete
	URL
		https://platform.clearblade.com/apidev/<collectionID>?query=<URLENCODEQUERYOBJ>
	TYPE
		DELETE
	Headers
		CLEARBLADE-APPKEY:<key>
		CLEARBLADE-APPSECRET:<secret>
